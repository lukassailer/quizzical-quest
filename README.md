# quizzical-quest

a trivia quiz game using:
- sbt
- scala.js
- scalajs-react
- npm
- vite
- the-trivia-api

## Prerequisites
- sbt
- Node (for npm)

## Installation
```
npm install
```

## Development
Start continuous compiler in sbt terminal:
```
~fastLinkJS
```
Start vite dev server in separate terminal:
```
npm run dev
```
