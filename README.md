# Quizzical Quest

A trivia quiz game made with:

- scalajs
- scalajs-react
- sbt
- npm
- vite

## Prerequisites

These need to be installed:

- sbt
- Node.js

## Installation

```bash
   npm install
   ```

## Development

In sbt:

```sbt
   ~fastLinkJS
   ```

In another terminal:

```bash
   npm run dev
   ```

## Production

```bash
   npm run build
   ```
